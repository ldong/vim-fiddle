## Instruction

1. To use, just clone this repository. `git clone https://github.com/ldong/vim-fiddle`

2. Set `Google Chrome` as your default browser on Mac

3. `./fiddle.sh`

4. Use ``<Leader>r`` to compile and open the file in browser or refresh it.
